# Conformance

Protocol conformance rules ensuring interoperability across independent implementations. Covers canonical encoding, hashing, signing, and test vectors.

- `canonical_cbor_rules.md` — Byte-level Canonical CBOR encoding rules for stable object IDs and signatures
- `reference_implementation.md` — Definition of the normative reference implementation (CBOR, BLAKE3, Ed25519)
- `test_vectors/` — Generated test vectors for conformance validation
