# Schemas

Data schemas for game objects including constructibles, entities, items, plots, and resources. Used by the construction system, gardening, and world simulation.

## Contents

- `constructibles.md` — Schema for constructible building objects (walls, furniture, utilities)
- `entities/` — Entity schemas (plants, substrates)
- `items/` — Item schemas (harvest outputs)
- `plots/` — Plot schemas (garden/farm plots)
- `resources/` — Resource schemas (nutrients, water)
