# engine-wgpu-shell

Custom Rust + wgpu runtime shell scaffold.

## Run

```bash
cargo run -p engine-wgpu-shell
```

## Controls

- `W/A/S/D`: move
- `Mouse`: look
- `Shift`: sprint
- `E`: gather wood
- `Q`: gather fiber
- `R`: craft filter kit
- `T`: treat water
- `F`: farm tick
- `C`: eat food ration

Window title shows live world/inventory/milestone state.
