# Project Universe Systems Overview

## Mission

Project Universe aims to end poverty through peaceful human unity by creating a free, open-source platform that serves as a comprehensive operating system in video game form. It combines real-life productivity tools with educational gameplay to achieve autonomous self-sustainability through universal homesteading.

## Core Philosophy

- **Reality + Fantasy Integration**: Tools that work in both real life and game contexts
- **Educational Focus**: Every system teaches practical, real-world skills
- **Self-Sustainability**: Complete independence through homesteading anywhere in the universe
- **Unified Platform**: Single application containing everything users need

## Development Guidelines

### Code Organization

- Each system in dedicated `.rs` file in `/source/`
- Data files in RON format in `/data/`
- Shared utilities in `/source/lib.rs`
- Widgets and UI components in `/source/widgets/`

### Data Management

- CSV files for tabular data
- RON files for complex configurations
- Version control for all data changes
- Data validation and integrity checks

### UI/UX Design

- Consistent theme system (`theme.ron`)
- Reusable widget library
- Accessibility-first design
- Cross-platform compatibility

### Testing & Quality

- Unit tests for all systems
- Integration testing for cross-system interactions
- Performance benchmarking
- User experience testing
