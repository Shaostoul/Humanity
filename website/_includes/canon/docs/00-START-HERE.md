# 00-START-HERE

If this is your first day in this repo, start here.

## 1) What this project is

Humanity is a modular ecosystem with two connected layers:

- **Humanity Platform**: practical education, simulations, communication, tools
- **Humanity Game Universe**: sci-fi/fantasy gameplay and story

These layers share core systems (math, physics, progression), but neither should hard-depend on the other.

Read next: [`01-VISION.md`](./01-VISION.md)

## 2) Ground rules

- Keep modules small and composable.
- Prefer plain Markdown docs and readable Rust over clever abstractions.
- Every folder should be understandable in a basic editor (Notepad++, terminal, Obsidian).
- Avoid hidden magic and undocumented side effects.
