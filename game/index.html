How It Works

1. You stream
Your camera/screen goes to the relay server.
2. Relay distributes
Server sends to the first viewers (has way more bandwidth than you).
3. Viewers seed
Each viewer shares chunks with others. More viewers = more seeds = better for everyone.
4. You control bandwidth
Set how much upload you want to contribute. Gaming? Slide it low. Idle? Crank it up.
