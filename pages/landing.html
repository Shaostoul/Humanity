Philosophy

What makes this different?

Built on principles, not profit. Every design choice serves the mission.

CC0 Public Domain

No permission needed

Every line of code is CC0. Fork it, modify it, sell it, teach from it. This belongs to everyone.

No Accounts

Cryptographic identity

Your Ed25519 key pair is your identity. No email, no password, no personal data collected. Ever.

E2E Encrypted

Private by default

DMs use ECDH key exchange with AES-256-GCM. Your notes use PBKDF2-derived keys. Only you can read your data.

Self-Hostable

Run your own

Single Rust binary, SQLite storage, no external dependencies. Deploy in minutes on any Linux box.

Federable

Connected but independent

Relay servers can link together. Your community stays sovereign while connecting to the wider network.

No Build Step

Plain JS, plain HTML

No npm, no webpack, no React. Just script tags and CSS custom properties. Anyone can read and modify the client.

Nobody owns this. Nobody can take it away. Your messages are locked with math only you can unlock. And you can run your own copy if you want.
Identity is Ed25519 key pairs stored in browser localStorage. DMs use X25519 ECDH for shared secrets + AES-256-GCM. Vault/notes encrypt with PBKDF2-SHA256 (600k iterations). Federation is server-to-server WebSocket relay.
